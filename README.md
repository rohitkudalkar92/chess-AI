# Chess AI

A chess engine with AI-powered move generation.

## Features

- Chess board representation and move validation
- AI opponent using minimax algorithm with alpha-beta pruning
- Interactive gameplay

## Getting Started

### Prerequisites

- Python 3.8+

### Installation

```bash
git clone https://github.com/<username>/chess-AI.git
cd chess-AI
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
